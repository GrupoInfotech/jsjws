jsjws

pure JavaScript implementation of JSON Web Signature

Download this project as a .zip file Download this project as a tar.gz file
JSON Web Signature(JWS) is a protocol for signing JSON data structure defined in [1] [2] and one of basic component for OpenID ABC or OpenID Connect. The 'jsjws'(JSON Web Signature JavaScript Library) is a pure open source free JavaScript implementation of it.

  • Sample Application for JWS signature validation

LIBRARY FEATURES

  • sign and verify a JSON Web Signature(JWS) of RSA algorithm
  • implemented by pure JavaScript
  • 'jsjws' will work on any modern browsers including smart phones
  • 'jsjws' supports 'ES256' and 'ES512' signature algorithms
  • 'jsjws' supports only ASCII characters in JWS Header and Payload
